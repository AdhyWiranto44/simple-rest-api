# Simple REST API

Project boilerplate untuk membangun REST API menggunakan Node.js, Express, dan Mongodb

Studi kasus: Pengelolaan blog

```
# Routes
GET      /api/posts
POST     /api/posts
GET      /api/posts/:slug
PUT      /api/posts/:slug
DELETE   /api/posts/:slug

```

## Cara penggunaan

```
# Install dependencies
npm install

# Run
npm start

```
